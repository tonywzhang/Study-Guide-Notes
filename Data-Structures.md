# Data Structure Notes

| Data Structure  | Insertion Time | LookUp Time|  Memory |
| ------------- | ------------- | -----------| ---------|
| Array | O(log(n)) | O(1) | O(n)|
| Linked Lists | O(1) | O(n) | O(n) |
| Binary Trees | O(log(n)) | - | O(n) |
| Binary Search Trees | O(log(n)) | O(log(n)) | O(n) |
| Set | O(1) | O(1) | O(n) |
| Map | | | |
